# Widen

Acquia Digital Asset Management (formerly Widen Collective) provides a REST API for managing digital assets, metadata, collections, embed codes, and asset distribution workflows.

The API supports asset search and discovery, metadata management, analytics, orders, products, webhooks, and workflow automation. Both v1 and v2 API versions are available, with v2 as the primary interface.
